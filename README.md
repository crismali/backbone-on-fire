backbone-on-fire
================

Supercharged Backbone

Development
-----------

Install dependencies
```sh
npm install -g gulp
npm install -g karma
npm install
```

Run the watching compiler
```sh
gulp
```

Run the specs
```sh
karma start karma.config
```

Install Example dependencies
```sh
cd example
bower install
```

Run the example site
```sh
node server.js
```
